Object-Goal Navigation (ObjectNav) requires an agent to autonomously explore an unknown environment and navigate toward target objects specified by a semantic label. While prior work has primarily studied zero-shot ObjectNav under 2D locomotion, extending it to aerial platforms with 3D locomotion capability remains underexplored. Aerial robots offer superior maneuverability and search efficiency,

中文介绍 提出AION，面向空中机器人的室内目标导航框架，采用双策略强化学习实现3D空间探索。相比2D平台，利用飞行器机动性显著提升搜索效率，填补零样本ObjectNav在空中的空白。
